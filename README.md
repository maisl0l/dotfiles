## dotfiles for arch

### archinstall configs:
```sh
wget https://raw.githubusercontent.com/maisl0l/dotfiles/main/archinstall/archinstall.sh
bash archinstall.sh
```

### install dotfiles:
```sh
cd /home/
git clone https://github.com/maisl0l/dotfiles
bash /home/dotfiles/pacman.sh && bash /home/dotfiles/updoot.sh
```
