## archinstall configs:
```sh
curl https://raw.githubusercontent.com/maisl0l/dotfiles/main/archinstall.sh > archinstall.sh
bash archinstall.sh
```

## install dotfiles:
```sh
cd
git clone https://github.com/maisl0l/dotfiles
bash /home/dotfiles/packages.sh && bash /home/dotfiles/updoot.sh
```
